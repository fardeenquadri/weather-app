# Weather App

A minimal static weather demo app built with HTML, CSS and JavaScript.

## Description

This project contains a simple front-end app that demonstrates fetching and displaying weather-related UI. It is a static site consisting of `index.html`, `style.css`, and `app.js`.

## Files

- [index.html](index.html): App entry point.
- [style.css](style.css): Styles for the UI.
- [app.js](app.js): Front-end JavaScript logic.

## Prerequisites

- A modern web browser.
- (Optional) Python or Node.js to serve files locally.

## Run locally

Open `index.html` directly in your browser (double-click the file), or run a simple local server:

Using Python 3:

```bash
cd d:\projects\weather-app
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

Using Node (http-server):

```bash
cd d:\projects\weather-app
npx http-server -p 8000
# then open http://localhost:8000
```

## Development

- Edit `app.js` to change behavior.
- Edit `style.css` to change styles.
